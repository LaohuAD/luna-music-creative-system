# 封面风格模板（供 `publish-cover-creation` Skill 调用）

本文件定义 `publish-cover-creation` Skill 内置的九种固定封面风格与一种用户自定义风格。

重点不是只告诉 AI “画风叫什么”，而是告诉它：

1. 这种风格的视觉气质是什么
2. 适合什么类型的标题
3. 标题应该用什么字体或艺术字策略
4. 标题如何嵌入画面
5. 画面中的哪些元素可以承担引导线功能
6. 最终提示词应该如何写得具体、完整、可执行
7. 标题在画面里应该占多大比例才足够抓眼
8. 人物是否需要更近的镜头、更夸张的姿势、更非日常的造型来打破平庸

所有示例提示词都默认是**中文提示词**。
凡是需要出现在画面中的文字信息，都必须使用英文双引号 `""` 包裹。

为了便于直接出图测试，下面九种固定风格的示例统一使用：
- **人物设定**：女性人物（参考图为女性形象）
- **标题内容**：`"luna音乐创作系统"`

## 1. 科技感未来主义

### 风格解释
强调未来、科技、数字化、系统感、秩序感。适合讲 AI、系统、流程、效率、升级、方法论这类内容。

### 适合的字体策略
- 科技感无衬线字体
- 玻璃质感字
- 金属立体字
- 发光边缘字

### 标题嵌入建议
- 适合做成**嵌入式**或**生成式标题**
- 标题可以像悬浮在界面中的信息层，或者像一个真实存在的数字装置
- 标题占比要明显偏大，至少成为人物之外的第二主体，而不是角落说明字

### 引导线建议
- 数据流
- 全息界面边框
- 声波走向
- 透视屏幕边缘
- 人物目光/手势指向标题

### 输出示例

- 适合做成**生成式标题**
- 标题可以直接成为街头招牌、电子屏、霓虹灯牌的一部分
- 标题一定不能太小，应该像一块真正能压住画面的霓虹主招牌

### 引导线建议
- 街道透视
- 霓虹灯带
- 建筑垂直线
- 反光地面
- 人物身体朝向

## 7. 极简主义

### 风格解释
强调克制、秩序、专业、高级感。适合系统、方法、观点、商业、设计感较强的内容。

- 少量异形结构字

- 留白边界
- 简洁图形方向
- 单一物体朝向

## 10. 用户自定义

### 风格解释
当用户有明确风格要求，而九种内置风格都不够贴切时，使用自定义风格。

### 使用规则
- `cover_style` 设置为 `自定义`
- 必须同时提供 `custom_style_description`
- Skill 需要优先遵循用户描述，再结合封面设计原理做结构优化

### 输出策略
即使是自定义风格，也仍然必须分析：
- 适合什么字体
- 标题如何嵌入画面
- 标题是否足够大
- 画面如何建立视觉中心
- 哪些元素承担引导线功能
- 是否需要更夸张的人物状态、更精致的妆造、更非日常的反差来增强传播性
- 不同比例下如何保持信息一致性

### 输入示例
```json
{
  "cover_title": "luna音乐创作系统",
  "cover_style": "自定义",
  "custom_style_description": "整体像高级科幻电影海报，使用女性人物形象，冷色调，标题像悬浮在空间中的金属结构，画面有很强的透视引导线"
}
```

## 使用提醒

调用本模板时，不要只套风格名称，要真正把下面这些一起想清楚：

1. 这个标题更适合规整字还是艺术字？
2. 标题够不够大，缩略图里能不能一眼读到？
3. 重点词该不该单独强化？
4. 标题是贴附在画面里，还是本身就是画面主体？
5. 人物是不是应该更近、更夸张、更有戏？
6. 服装、首饰、妆容能不能承担额外记忆点？
7. 观众第一眼先看哪里，第二眼如何落到标题？
8. 哪些线条、光区、人物动作、场景透视在帮标题服务？
9. 标题普通时，画面是否足够不普通？

风格只是外壳，**标题、字体、视觉中心、引导线、人物状态、反差感，才决定封面有没有真正的抓力。**
