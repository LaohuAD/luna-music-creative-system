# 歌词封面设计指南

本文件继承 `publish-cover-creation` 的封面设计原则，并针对“歌词封面 / 单曲海报”做定向补充。

## 1. 本质

歌词封面不是教程封面，也不是资讯封面。
它更接近：
- 单曲封面
- EP / Demo 海报
- 情绪化视觉入口
- 用一张图把一首歌的核心意境钉住

所以它的首要任务不是解释清楚所有信息，而是：

1. 抓住这首歌词最有画面感的那一层
2. 让观众产生情绪共鸣或想象空间
3. 让歌名成为海报中的主视觉标识
4. 在 1:1 的小尺寸下仍然保持精致、清晰、耐看

## 2. 1:1 比例原则

1:1 比例更像专辑封面或流媒体封面，因此：

- 视觉中心必须更集中
- 标题与主体距离不能太散
- 背景层次可以有，但不能过碎
- 人物如果出现，通常优先近景、半身或具有清晰姿态的中景
- 远景大场面只有在歌词本身非常依赖空间感时才使用

## 3. 人物优先原则

本 skill 默认优先考虑人物，因为人物最容易承载情绪与故事。
但“有人物”不等于“硬塞一个人头”。

判断标准：
- 这首歌词有没有明确的“谁”
- 情绪是内心独白、对望、离别、等待、奔跑、沉沦，还是纯意象
- 人物是更适合近距离承载情绪，还是更适合放在场景中成为叙事坐标

可选方式：
- 人物特写：适合高情绪浓度
- 半身近景：适合兼顾表情与姿态
- 中景人物：适合强调动作和环境关系
- 远景人物：适合孤独、渺小、命运感
- 无人物强场景：只在场景本身足够强时使用

## 4. 画面提炼方法

从歌词中优先提炼以下四类信息：

1. **核心情绪**：遗憾、希望、失真、炽热、孤独、失控、释然
2. **关键意象**：雨夜、车站、海面、灯塔、窗边、旧房间、霓虹街头、月光、火焰
3. **动作瞬间**：回头、奔跑、拥抱、松手、凝视、坠落、站在风里
4. **空间关系**：一个人 vs 两个人，人物与城市，人物与房间，人物与自然

优先抓“最值得被画出来的一幕”，而不是平均分配整首歌词内容。

## 5. 歌名标题原则

歌名不是封面下方说明文字，而是海报主标题。

必须做到：
- 用艺术字呈现
- 字体材质和画面情绪匹配
- 大小足够支撑缩略图阅读
- 与人物/场景/光线/引导线统一
- 不和主体抢戏，但也不能弱成附属说明

常见处理方式：
- 发光艺术字
- 金属字
- 手写质感字
- 书法字
- 霓虹字
- 玻璃字
- 纸感拼贴字
- 雾化光影字

## 6. 海报感判断

一张歌词封面要像海报，而不是普通插图。

至少应满足：
- 有明确主视觉
- 有画面层次
- 有光线方向
- 有完成度高的标题设计
- 有情绪氛围
- 有可被记住的瞬间

## 7. 分析输出要求

本 skill 不能只给 prompt 结论，必须给出分析过程，至少解释：

- 这首歌词的核心情绪是什么
- 为什么抓这个画面，而不是别的画面
- 为什么人物应该出现 / 不应该出现
- 为什么选这个景别
- 为什么歌名要用这种艺术字
- 为什么标题放在这里
- 为什么 1:1 比例下这样构图更成立
