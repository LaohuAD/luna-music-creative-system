# 封面风格模板（供 `music-cover-creation` Skill 调用）

本文件复用并继承 `publish-cover-creation` 的封面风格方法，但服务对象从“自媒体内容标题”改为“歌词封面”。

重点不是单纯做一张好看的图，而是：

1. 把歌词转成可被看见的画面
2. 让封面像一张真正能打动人的单曲海报
3. 强化人物、场景、情绪、光线、构图与歌名之间的统一关系
4. 让标题文字真正成为海报的一部分，而不是后贴标签
5. 在 1:1 比例中依然保留强烈的画面中心和缩略图识别度

本参考文件默认继承 `publish-cover-creation` 中关于风格、字体、标题嵌入、引导线、人物镜头与非日常感的标准。执行时优先读取同目录下 `self-media-cover-design-guide.md`，并结合源 skill `publish-cover-creation` 的同名参考理解风格逻辑。

## 额外约束

- 默认目标比例是 **1:1**
- 封面优先服务“歌曲意境”而非“知识表达”
- 画面中尽量出现人物，但也允许在特定歌词下以强场景为主
- 人物可以是特写、近景、半身、中景、远景，但必须有明确叙事作用
- 歌名必须出现在画面中，并且必须是 **艺术字**
- 标题必须足够大，且与画面和谐统一，像真正的海报主标题
- 封面需要具备海报级精美度、氛围感和完成度

## 风格使用原则

调用风格模板时，不要机械套模板。必须先判断：

1. 这首歌词更偏叙事、情绪、幻想、都市、治愈、暗黑还是实验感
2. 封面的画面中心应该是人物、关系、动作，还是某个场景瞬间
3. 哪一句歌词最值得被转成画面
4. 歌名更适合做成哪种艺术字材质与结构
5. 1:1 比例下，哪些元素必须删减，哪些必须被放大

## 推荐默认风格映射

- 抒情 / 治愈 / 回忆 → 治愈温暖、光影写实
- 宏大 / 未来 / 系统 / 梦境 → 科技感未来主义、赛博朋克、复古蒸汽波
- 东方 / 诗性 / 古风 / 留白意境 → 国潮水墨
- 冲突 / 深夜 / 黑暗 / 隐秘 → 暗黑神秘
- 年轻 / 轻盈 / 可爱 / 活泼 → 清新卡通
- 高级 / 冷静 / 电影感 / 品牌化 → 极简主义、光影写实

## 标题规则

凡是需要出现在画面中的文字，都必须使用英文双引号 `""` 包裹。

歌名标题必须满足：
- 是艺术字，不得只是普通排版
- 是画面构成的一部分
- 在 1:1 缩略图中依然清晰
- 与歌词的情绪、风格、光线和材质统一
