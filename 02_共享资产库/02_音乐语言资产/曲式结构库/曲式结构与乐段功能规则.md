# 曲式结构与乐段功能规则

本文件是老胡音乐关于先曲后词、曲式骨架、段落功能和编曲空间的专项规则库。

它比 `laohu_music_structure` skill 更细，负责补充曲式裁决、段落功能、hook space、人声预留空间和纯音乐 style prompt 的质量校验。

老胡音乐当前流程不是完全先词后曲，也不是完全先曲后词，而是：

```text
先确认题眼和副歌核心句
→ 再用曲式和风格框住情绪
→ 再按这个骨架填词
```

## 1. 曲式与风格阶段的任务

曲式不是格式，风格不是审美标签。

这一阶段要回答：

- 什么声音容器最能放大题眼？
- 什么曲式能逼出副歌核心句？
- 哪个乐段负责叙事，哪个乐段负责收紧，哪个乐段负责释放？
- 哪里要留出 hook space？
- 哪里要给人声预留气口？
- 编曲要在哪些地方让位给歌词？

## 2. 常见曲式功能

### Intro

作用：

- 建立空间、情绪和听觉入口。
- 不抢副歌核心句。
- 可以留下一个可截取的短视频入点。

### Verse

作用：

- 让人物进入。
- 给出具体处境、动作和关系压力。
- 不负责解释完整背景。

判断：

```text
Verse 是否让副歌更必然？
如果删掉 Verse，副歌是否会变成空喊？
```

### Pre-Chorus

作用：

- 收紧情绪。
- 把主歌里的生活压力推到副歌前。
- 让副歌不是突然喊出来。

常见做法：

- 句长变短。
- 和声或节奏逐步抬升。
- 留出入副歌前的停顿。
- 把问题逼到“不得不唱”的位置。

### Chorus

作用：

- 兑现副歌核心句。
- 释放情绪峰值。
- 承担传播资产。

硬门槛：

- 短、准、可复唱。
- 至少有一句能被记住。
- 不要只是把主歌意思说大声。

### Bridge

作用：

- 翻面、反问、视角移动、代价暴露或关系重估。
- 不是换一组漂亮句子。

可用方向：

- 从“我以为”翻到“我后来发现”。
- 从对别人说，翻到对自己说。
- 从个人情绪，翻到更大的命运或时代。
- 从不甘，翻到接受，但不能鸡汤。

### Outro

作用：

- 回收题眼。
- 给余波。
- 留下可以循环的尾句。

禁区：

- 不要作文总结。
- 不要突然升华。
- 不要把副歌解释一遍。

## 3. 能量曲线

曲式输出必须说明能量曲线：

```text
Intro：低能量，建立空间
Verse：信息清楚，近景
Pre：收紧，压迫感上升
Chorus：打开，核心句释放
Verse 2：补新证据，不重复
Bridge：情绪翻面或关系重估
Final Chorus：重复但更重
Outro：回收，留白
```

不能全程同一种强度。情绪没有起伏，听众很难记住副歌。

## 4. 纯音乐 style prompt 原则

曲式阶段输出的是纯音乐骨架提示词，不是最终人声版本。

可以写：

- subgenre。
- groove。
- BPM。
- drums / bass / guitar / piano / synth。
- arrangement density。
- hook space。
- vocal pocket。
- leave room for lead vocal。
- dynamic arc。
- space / texture。

不要写：

- 具体男声 / 女声画像。
- 具体唱法。
- 每句歌词的演唱控制。
- generic pop ballad。
- cinematic emotional song。
- piano ballad 这种中心值套话。

## 5. 主风格与辅助风格

必须裁决主风格。辅助风格只能服务局部，不允许平均混合。

错误：

```text
流行 + 民谣 + R&B + 电子 + 国风，全部都要。
```

正确：

```text
主风格：华语流行 R&B。
辅助质感：Verse 用 close-mic acoustic guitar 增加亲近感，Chorus 保持 R&B groove 和宽声场。
```

## 6. 风格不 generic 检查

最终曲式风格输出必须具体到：

- 细分音乐风格。
- groove / BPM。
- 风格内乐器。
- 编曲密度。
- 空间质感。
- 情绪弧线。
- 后续填词空间。

如果只写：

```text
抒情流行，钢琴，温暖，感人
```

视为失败。

## 8. 输出前自检

- 曲式有没有服务题眼？
- 副歌核心句出现位置是否明确？
- Verse / Pre / Chorus / Bridge / Outro 是否各有任务？
- 主风格是否明确？
- 是否避免 generic 中心值？
- 是否给后续填词留了气口和句长空间？
