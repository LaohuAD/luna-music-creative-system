# 灵感关键词到音乐创作维度映射规则

本文件是老胡音乐把灵感、关键词、故事和人物材料映射到音乐创作维度的专项规则库。

它比 `laohu_music_topic` skill 的常规判断更细，负责在灵感模糊、材料较多或方向摇摆时，帮助 agent 进一步判断题材、人称、场景、核心物象、情绪、风格倾向、副歌功能和禁用方向。当前项目只通过对话、口述、文件材料和老胡反馈来判断。

## 1. 映射目标

当老胡给出一句灵感、一段故事、一个人物或一组关键词时，不能只把它整理成“主题”。要把它映射成后续音乐创作真正需要的维度：

```text
题材类型
→ 嘴替对象
→ 说话人称
→ 场景入口
→ 核心物象
→ 核心情绪
→ 表达方式
→ 主风格倾向
→ 副歌功能
→ 禁用方向
```

## 2. 题材类型映射

常见关键词：

- 家人、父母、爷爷、奶奶、老伴、孩子：亲情 / 代际 / 亏欠。
- 分手、暧昧、冷暴力、沉默、错过：亲密关系 / 遗憾 / 没说出口。
- 城市、街头、华强北、视频、走红、帮扶：社会观察 / 当代现实。
- 坚持、命运、苦难、年、人生：人生叙事 / 命运感。
- 权力、规则、防御、拆穿、不服：态度表达 / 说唱 / 摇滚 / 社会观察。

注意：关键词只提供线索，不是最终题材。必须回到“这首歌替谁说话”。

## 3. 人称和说话位置

默认判断：

- 第一人称：适合亲历、独白、心事、没说出口的情绪。
- 第二人称：适合对某个人说、质问、告别、挽留、拆穿。
- 第三人称：适合人物故事、社会观察、历史人物、旁观叙事。

判断问题：

- 谁有资格说这句话？
- 谁能看见这些材料？
- 谁不能越权替别人说内心？
- 听众站在谁的位置听？

如果人称不稳，后续歌词会散。

## 4. 场景入口

场景入口不是背景板，而是第一段歌词从哪里开门。

常见入口：

- 动作入口：删掉、忍住、回头、还钱、等、躲、没说完。
- 空间入口：街头、房间、楼道、车里、摊位、门口。
- 物件入口：票据、灯、饭、药、花、旧衣服、聊天框。
- 时间入口：多年后、凌晨、换季、最后一次、某个节日前。

禁区：

- 不要用“深夜、地铁、没回消息、妈妈问到家了吗”这类高频 AI 素材做默认高级示范。
- 如果这些素材来自老胡真实输入，可以用；否则不要自动套。

## 5. 核心情绪映射

同一灵感可能有多个情绪，不要只抓最表层。

- 不甘：适合 Rap、摇滚、强节奏流行。
- 遗憾：适合流行、民谣、R&B。
- 想念：适合流行、民谣、国风，但要避免泛泛温柔。
- 亏欠：适合民谣、流行、现实叙事。
- 清醒 / 拆穿：适合 Rap、摇滚、冷感电子。
- 治愈：必须有伤口和代价，不要直接写“会好起来”。

## 6. 表达方式映射

- 白描叙事：适合现实人物、亲情、街头观察。
- 口语独白：适合第一人称心事。
- 对话推进：适合第二人称、质问、告别。
- 借物抒情：适合有稳定核心物象的题材。
- Rap bar：适合态度、社会观察、反击、争议。
- 冷感碎片：适合电子、疏离、循环情绪。
- 国风凝练：适合历史、命运、人物抉择。

## 7. 主风格倾向映射

风格不能只看情绪，要同时看：

```text
人物身份
→ 情绪强度
→ 语言密度
→ 传播场景
→ 副歌核心句形态
```

判断示例：

- 副歌核心句短、准、容易传唱：优先流行。
- 需要讲人物和生活处境：民谣可选，但不能默认。
- 态度强、有观点刀口：Rap / 摇滚优先。
- 私密、贴耳、拉扯：R&B 优先。
- 冷感、循环、身体节奏：电子优先。
- 历史、命运、凝练意象：国风 / 古风可选。

## 8. 副歌功能映射

副歌不一定都是“情绪爆发”，它可能承担不同功能：

- 替听众说出口。
- 把主歌细节压成一句判断。
- 把关系代价唱出来。
- 把标题句变成传播句。
- 把人物从忍住推到承认。
- 把不甘推到反击。

副歌核心句不是越狠越好，而是越准越好。

## 9. 禁用方向映射

每次选题题眼阶段都要给“不建议走的方向”。

常见禁用：

- 不要写成营销口号。
- 不要写成公益宣传。
- 不要写成城市宣传片。
- 不要把现实人物写成苦情消费。
- 不要把治愈写成廉价暖词。
- 不要把 Rap 修成民谣。
- 不要把国风写成符号堆砌。
- 不要用 AI 高频苦情素材冒充高级。

## 10. 输出前自检

- 是否已经从关键词映射到了音乐创作维度？
- 是否明确嘴替对象和说话位置？
- 是否有场景入口和核心物象？
- 是否判断了主风格倾向，而不是默认流行或民谣？
- 是否给出副歌功能？
- 是否明确禁用方向？
