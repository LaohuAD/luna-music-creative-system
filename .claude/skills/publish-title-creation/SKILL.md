# `publish-title-creation` Skill 使用指南

## 1. Skill 核心能力

本 Skill 旨在帮助用户为自媒体内容创建高质量、高吸引力的标题和封面标题。它不仅提供最终结果，还会深入分析生成过程，确保标题既符合平台特性，又能有效吸引目标受众。

### 1.1 标题生成
- **平台适配**：针对小红书、抖音、视频号、B站、公众号生成定制化标题。
- **爆款潜力**：运用情绪驱动、钩子法则、中文语境模板等，提升标题的点击率和传播力。
- **详细分析**：提供每个标题的生成思路、情感策略、平台适配理由和预期效果，并使用多维度评分体系进行评估。

### 1.2 封面标题与设计指导
- **通用封面标题**：生成一个可在所有平台通用的精炼封面标题。
- **设计理念描述**：根据封面标题，提供封面设计方向的描述，包含视觉元素、风格、色彩搭配等建议。
- **生成封面提示词**：结合设计理念，给出可用于图像生成工具的提示词，便于后续生成封面。

## 2. 工作流程

用户提供创作灵感（`inspiration`）和可选的目标平台列表（`platforms`）。

1. **内容核心提取**：AI 首先分析 `inspiration`，理解内容的核心价值、主要受众和潜在的情绪点。
2. **标题多平台生成**：
   - 对于每个指定平台，AI 根据 `references/self-media-title-guide.md` 中的规则，包括字数限制、平台风格、情绪驱动、中文模板等，生成 3-5 个候选标题。
   - 对每个标题进行多维度评分（准确性、情绪触发、目标匹配、平台适配、信息密度）。
   - 选出最高分的一个标题，并给出详细的分析过程和评分理由。
3. **通用封面标题生成**：AI 从内容核心中提炼 3-8 个字，生成一个简洁、有吸引力且普适性强的封面标题。
4. **封面设计理念与提示词生成**：
   - AI 根据封面标题，结合内容的视觉化潜力，描述一个封面设计理念（例如：使用鲜明的色彩对比、具象化抽象概念、制造视觉冲击等）。
   - 进一步，将设计理念转化为具体的图像生成提示词，考虑风格（写实、插画、卡通）、构图、主色调等。
5. **结果呈现**：以结构化的方式返回所有平台的标题、封面标题、封面设计理念、提示词以及详细的分析过程。

## 3. 输入参数

- `inspiration` (string, required): 用户的创作灵感。可以是一篇文章的草稿、一个段落、一句话、一个想法，甚至是一个已有的标题。
- `platforms` (array of string, optional): 目标平台列表。可选项：`xiaohongshu`, `douyin`, `video_channel`, `bilibili`, `wechat_official_account`。如果未提供，则默认为所有五个平台生成标题。

## 4. 输出格式

本 Skill 将返回一个结构化的 JSON 对象，包含以下字段：

- `platform_titles` (array):
  - `platform` (string): 平台名称。
  - `title` (string): 生成的标题。
  - `analysis` (string): 标题生成过程的详细分析，包括：内容核心提取、目标读者分析、情绪策略、钩子类型、平台适配理由。
  - `score` (object): 标题的五维评分（准确性、情绪触发、目标匹配、平台适配、信息密度，每项0-10分）。
  - `recommendation` (string): 推荐理由或改进建议。
- `cover_title` (string): 适用于所有平台的通用封面标题。
- `cover_design_concept` (string): 封面设计理念的描述，解释封面标题与内容的视觉配合。

并且将返回的内容保存成为markdown格式，保存在你的工作区：workspace/projects/luna-music-creative-system/publish-title-creation/{文件名}-{日期}.md

## 5. 引用文档

本 Skill 的所有标题生成和评估逻辑，均严格遵循以下参考文档：

- `references/self-media-title-guide.md`
  - 该文档详细阐述了标题核心原则、平台字数限制、风格差异、情绪驱动、中文模板、红线边界以及标题质量评判体系。
- `references/self-media-cover-design-guide.md`
  - 该文档详细阐述了封面核心原则、构图、颜色、人物表情、文字设计、场景道具、图文配合、平台规范和评判体系。

## 6. 使用示例

假设用户想为一个关于"AI如何辅助音乐创作"的灵感生成标题。

```json
{
  "inspiration": "AI工具越来越成熟，普通人也能用AI创作音乐了，这让很多创作者既兴奋又有点焦虑。我写了一篇文章，想介绍AI在音乐创作中的具体应用，以及如何利用AI提高创作效率，避免被AI取代。",
  "platforms": ["xiaohongshu", "douyin"]
}
```

## 7. 边界与注意

- 本 Skill 侧重于标题和封面设计理念的生成，不直接生成图像。生成的提示词需用户自行调用图像生成工具。
- AI 的分析过程将尽可能详细，但最终决策权仍在用户。
- 遵循 `references/self-media-title-guide.md` 中的"红线与边界"章节，避免生成误导性或过度承诺的标题。
